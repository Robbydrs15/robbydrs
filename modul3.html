Pengenalan HTML & CSS

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dasar halaman web. HTML menggunakan berbagai tag untuk mendefinisikan elemen-elemen seperti heading, paragraf, gambar, dan tautan. Setiap halaman web yang kita lihat di internet dibangun dengan fondasi HTML yang kuat.

Memahami CSS

CSS (Cascading Style Sheets) adalah bahasa styling yang digunakan untuk mengatur tampilan dan format elemen HTML. Dengan CSS, kita dapat mengontrol warna, ukuran, jarak, tata letak, dan berbagai aspek visual lainnya dari sebuah halaman web. CSS memungkinkan pemisahan antara konten (HTML) dan presentasi, membuat kode lebih terorganisir dan mudah dikelola.
