    Menampilkan Data dalam JavaScript

    1.metode document.write() dapat digunakan untuk menulis konten langsung ke dokumen HTML, namun ini jarang digunakan dalam pengembangan modern karena dapat menimpa konten yang ada. 2.properti innerHTML memungkinkan kita untuk memasukkan atau mengubah konten HTML di dalam elemen tertentu, memberikan fleksibilitas dalam manipulasi DOM. 3. metode appendChild() digunakan untuk menambahkan elemen baru ke dalam DOM, memungkinkan pembuatan dan penyisipan elemen secara dinamis. Metode ini sering digunakan bersama dengan createElement() untuk membuat struktur HTML yang kompleks secara programatik. Setiap metode memiliki kegunaannya sendiri, tergantung pada kebutuhan spesifik dan konteks pengembangan web.
